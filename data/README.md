## Формат данных

```json
{
  "id": "уникальный идентификатор",
  "email": "email@example.com",
  "password": "хешированный пароль (bcrypt)",
  "name": "Имя пользователя",
  "role": "admin | portfolio_manager | project_manager | viewer",
  "createdAt": "ISO дата создания"

```

## Автоматическая инициализация

При первом запуске приложения тестовые пользователи создаются автоматически через функцию `initializeTestUsers()` в `lib/db/users.ts`.
